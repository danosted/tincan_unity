## Quick Reference

### AI Can Do
✅ Implement features  
✅ Write documentation  
✅ Review code  
✅ Create automation scripts  
✅ Analyze architecture  
✅ Suggest improvements  

### AI Cannot Do
❌ Make game design decisions  
❌ Test on real hardware  
❌ Commit code without approval  
❌ Override human preferences  
❌ Make binding decisions about project direction
